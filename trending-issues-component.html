🔥 Trending Issues Widget

Live demonstration of the trending issues component

📋 How to Integrate This Component

Step 1: Add the CSS and JavaScript files to your HTML:

<link rel="stylesheet" href="trending-issues-styles.css"> <script src="trending-issues.js"></script>

Step 2: Add a container div where you want the widget to appear:

<div id="trending-issues-container"></div>

Step 3: The widget auto-initializes when the DOM is ready. No configuration needed!

Optional: Customize the number of issues displayed (default is 5):

// Show 10 trending issues instead new TrendingIssuesWidget("trending-issues-container", 10);

🎨 Features

  • ✨ Modern card-based layout with smooth animations
  • 🎯 Color-coded priority badges (Red/Yellow/Green)
  • 📱 Fully responsive design for all devices
  • ⚡ Loading skeleton animation for better UX
  • 🔄 Automatic error handling with retry button
  • 🎭 Beautiful gradient header with pulsing fire emoji
  • 📊 Shows votes, category, location, and priority

🔗 API Integration

The component fetches from: /api/issues/trending?limit=5

Make sure your backend API is running and accessible at this endpoint.
